The Extensible Devops Framework For The Complex Enterprice

CodeOnTap provides a best practice devops mode for enterprises that

  • Use laaS cloud but must keep somethings on premise
  • Use open source but also support and customise COTS products
  • Use offsite freelancers for capacity but want control with in-house IT
Get The Framework

Your Favorite Tools And Services

Extensible to support any cloud or enterprise application lifecycle management tools and IaaS platforms. But currently we support github, Docker, Jenkins, ElasticSearch, and Amazon Web Services

Our Conventions, Patterns & Scripts

Leverage our hard learned conventions and scripts for infrastructure as code, automated builds, docker container deployments, environment configuration, and secure credential management

A 90 Second Overview Of How It Works

  • Secure

    Enterprise security practices such as encryption of all credentials used in configuration files.

  • Flexible

    Deploy custom code or COTS apps to the cloud or on-premise so long as they can be containerised.

  • Proven

    In use by some large Corporations and Government Agencies.
